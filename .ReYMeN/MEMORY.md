# ReYMeN Hafıza Defteri

Bu dosya ajan tarafından otomatik güncellenir. Önemli bulgular, tercihler ve bağlam burada birikir.

## Sistem Tercihleri
- Birincil LLM: LM Studio (localhost:1234) — Ollama kaldırıldı
- Model: cognitivecomputations.dolphin3.0-llama3.1-8b
- Dil: Türkçe tercih edilir, kod İngilizce
- Onay: ReYMeN_ONAY_ISTE=false (sessiz mod)

## Proje Bağlamı
- ReYMeN, Windows 11'de çalışan otonom AI ajandır
- ReAct döngüsü: Düşün → Eylem → Gözlem → Tekrar
- Beceriler otomatik kristalleşir (closed_learning_loop.py)
- Vektörel hafıza ChromaDB ile (vektor_hafizasi/)
