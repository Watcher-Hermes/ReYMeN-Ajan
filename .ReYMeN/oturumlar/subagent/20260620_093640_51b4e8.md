# Python'da Async/Await — Detaylı Açıklama ve Kod Örnekleri

Python'daki `async`/`await` mekanizması, **eşzamanlı (concurrent)** çalışmayı sağlayan bir yapıdır. Altında üç temel kavram yatar:

1. **Coroutine** — `async def` ile tanımlanan, askıya alınıp devam ettirilebilen özel bir fonksiyon
2. **Awaitable** — `await` anahtar sözcüğü ile beklenebilen nesne (coroutine, Task, Future)
3. **Event Loop** — Coroutine'leri zamanlayıp yöneten ana döngü

---

## 1. Coroutine Temelleri

```python
import asyncio

# async def → bir coroutine fonksiyonu tanımlar
async def say_hello():
    print("Merhaba")
    await asyncio.sleep(1)    # 1 saniye bekle (bloklamaz!)
    print("Dünya")
    return "Bitti"

# Coroutine çağrılır ama hemen çalışmaz — bir coroutine nesnesi döner
coro = say_hello()
print(type(coro))  # <class 'coroutine'>

# Çalıştırmak için:
#   a) asyncio.run(coro)      — event loop'u başlatır
#   b) await coro             — başka bir coroutine içinde
result = asyncio.run(say_hello())
print(result)  # "Bitti"
```

> **Önemli:** `say_hello()` çağrısı fonksiyonu **çalıştırmaz**, sadece bir coroutine nesnesi oluşturur. Gerçek çalışma event loop'a teslim edilene kadar başlamaz.

---

## 2. Event Loop — Kalbin Atışı

Event loop, yapılacak işleri (coroutine'leri, callback'leri, I/O olaylarını) kuyruğa alır ve sırayla işletir. Bir coroutine `await` ile askıya alındığında, event loop başka bir coroutine'e geçer.

```python
import asyncio

async def task(name, duration):
    print(f"[{name}] Başlıyorum, {duration}s çalışacağım")
    await asyncio.sleep(duration)  # Burada event loop başka işe geçer!
    print(f"[{name}] Tamamlandı")
    return name

async def main():
    # 3 coroutine'i aynı anda başlat
    results = await asyncio.gather(
        task("A", 3),
        task("B", 1),
        task("C", 2),
    )
    print(f"Sonuçlar: {results}")

asyncio.run(main())
```

**Çıktı:**
```
[A] Başlıyorum, 3s çalışacağım
[B] Başlıyorum, 1s çalışacağım
[C] Başlıyorum, 2s çalışacağım
[B] Tamamlandı        ← 1 saniye sonra
[C] Tamamlandı        ← 2 saniye sonra
[A] Tamamlandı        ← 3 saniye sonra
```

> Toplam süre ≈ 3 saniye (sırayla çalıştırsaydık 6 saniye olurdu). `asyncio.sleep()` anında event loop'a kontrolü geri verir, böylece diğer coroutine'ler çalışabilir.

## 3. await: "Burada bekle, ama bloklama"

`await`, Python'a şunu söyler: *"Bu işlemin sonucunu bekliyorum, ama bu sırada event loop başka coroutine'leri çalıştırabilir."*

```python
import asyncio

async def fetch_data(url):
    print(f"İndiriliyor: {url}")
    await asyncio.sleep(2)  # Gerçek hayatta: aiohttp.get(url) vb.
    return f"{url} verisi"

async def main():
    # Sequential (sıralı) — toplam 4 saniye
    r1 = await fetch_data("site1.com")
    r2 = await fetch_data("site2.com")
    print(r1, r2)

    # Concurrent (eşzamanlı) — toplam 2 saniye
    r1, r2 = await asyncio.gather(
        fetch_data("site1.com"),
        fetch_data("site2.com"),
    )
    print(r1, r2)

asyncio.run(main())
```

> `await` olmadan coroutine çalışmaz. `await` olmazsa "coroutine never awaited" uyarısı alırsınız.

## 4. Task: Coroutine'i "Arka Planda" Çalıştır

`asyncio.create_task()` bir coroutine'i alır, hemen çalıştırmaya başlar ve bir `Task` nesnesi döner. Task'ler sayesinde birden çok işi aynı anda yürütebilirsiniz.

```python
import asyncio

async def worker(n):
    for i in range(3):
        await asyncio.sleep(0.5)
        print(f"Worker {n}: adım {i}")
    return f"Worker {n} bitti"

async def main():
    # Task'leri oluştur — hemen çalışmaya başlarlar
    t1 = asyncio.create_task(worker(1))
    t2 = asyncio.create_task(worker(2))

    print("Task'ler başlatıldı, başka iş yapıyorum...")
    await asyncio.sleep(0.3)
    print("Hâlâ devam...")

    # Task'lerin bitmesini bekle
    r1 = await t1
    r2 = await t2
    print(r1, r2)

asyncio.run(main())
```

**Çıktı (zaman akışı):**
```
Task'ler başlatıldı, başka iş yapıyorum...
Worker 1: adım 0
Worker 2: adım 0
Hâlâ devam...
Worker 1: adım 1
Worker 2: adım 1
Worker 1: adım 2
Worker 2: adım 2
Worker 1 bitti
Worker 2 bitti
```

## 5. Gerçek Hayat Örneği: HTTP İstekleri

```python
import asyncio
import aiohttp  # pip install aiohttp

async def fetch(session, url):
    async with session.get(url) as response:
        return await response.text()

async def main():
    urls = [
        "https://httpbin.org/delay/2",  # 2 saniyelik gecikme
        "https://httpbin.org/delay/1",
        "https://httpbin.org/delay/3",
    ]

    async with aiohttp.ClientSession() as session:
        tasks = [asyncio.create_task(fetch(session, url)) for url in urls]
        results = await asyncio.gather(*tasks)

    print(f"{len(results)} sayfa indirildi")
    print(f"İlk sayfa uzunluğu: {len(results[0])} karakter")

asyncio.run(main())
```

> `asyncio.gather()` ile 3 istek eşzamanlı gider. Toplam süre ≈ 3 saniye (en yavaş olanın süresi kadar). Sırayla gitseydi ≈ 6 saniye olurdu.

## 6. Kilit Kavramlar Özeti

| Kavram | Ne İşe Yarar? |
|---|---|
| **`async def`** | Coroutine (askıya alınabilir fonksiyon) tanımlar |
| **`await`** | Coroutine'i event loop'a bırakana kadar bekler |
| **`asyncio.run()`** | Event loop'u başlatır, ana coroutine'i çalıştırır |
| **`asyncio.create_task()`** | Coroutine'i arka planda çalıştırır |
| **`asyncio.gather()`** | Birden çok awaitable'ı eşzamanlı bekler |
| **`asyncio.sleep()`** | Bloklamayan bekleme (simülasyon için kritik) |
| **`async with`** | Async context manager (aiohttp session, dosya) |
| **`async for`** | Async iterator üzerinde döngü |

## 7. Yaygın Hatalar

```python
# HATA 1: await unutmak
async def wrong():
    asyncio.sleep(1)  # ❌ Çalışmaz! await gerekli
    # RuntimeWarning: coroutine 'sleep' was never awaited

# HATA 2: Bloklayıcı kod kullanmak
async def blocking():
    import time
    time.sleep(5)  # ❌ Event loop'u DONDURUR!
    # Bunun yerine: await asyncio.sleep(5)

# HATA 3: Coroutine'i direkt çağırmak
def run():
    my_coro()  # ❌ Sadece coroutine objesi döner, çalışmaz
    await my_coro()  # ❌ await, sadece async fonksiyon içinde kullanılır
```

## 8. Önemli Uyarı: Threading Değil!

Async/await **paralel değil, eşzamanlı (concurrent)** çalışır. Tek bir thread'de, gönüllü olarak kontrolü bırakma (cooperative multitasking) ile çalışır. CPU-bound işlerde (hesaplama, döngü) async **işe yaramaz** — bunun için `multiprocessing` veya `concurrent.futures.ProcessPoolExecutor` gerekir.

```python
# CPU-bound → async faydasız
async def cpu_heavy():
    total = 0
    for i in range(10**7):  # Bu döngü asla await yapmaz
        total += i
    return total
# 👆 Event loop, bu coroutine bitene kadar başka hiçbir şey çalıştıramaz!
```

**Async'in güçlü olduğu yerler:** I/O-bound işlemler — HTTP istekleri, dosya okuma, veritabanı sorguları, ağ çağrıları.

## Özet

- **async/await** = event loop üzerinde çalışan, gönüllü geçişli eşzamanlılık
- `await` her zaman event loop'a "şimdi başkası çalışsın" fırsatı verir
- `asyncio.gather()` ve `create_task()` ile eşzamanlı çalıştırma
- Tek thread'li, I/O-bound işler için idealdir
- CPU-bound işlerde kullanmayın
