# Python Decorator (Dekoratör) Nedir?

Python'da **decorator** (dekoratör), mevcut bir fonksiyonun veya sınıfın davranışını, kaynak kodunu değiştirmeden *genişleten* veya *değiştiren* özel bir yapıdır. Temelinde, bir fonksiyonu parametre olarak alıp onu saran (wrapping) başka bir fonksiyon döndüren **higher-order function**'dır.

Dekoratörler `@` sembolü ile kullanılır ve fonksiyon tanımının hemen üstüne yazılır.

---

## 1. Neden Dekoratör Kullanırız?

Gerçek hayatta sıkça karşılaşılan bazı tekrarlayan desenler vardır:

- **Loglama** — Bir fonksiyon çağrıldığında log yazmak
- **Yetkilendirme (Authorization)** — Kullanıcının erişim yetkisini kontrol etmek
- **Performans ölçümü** — Fonksiyonun ne kadar sürede çalıştığını ölçmek
- **Önbellekleme (Caching)** — Sık kullanılan sonuçları saklamak
- **Doğrulama (Validation)** — Girdi parametrelerini kontrol etmek

Dekoratörler olmadan bu kodu her fonksiyona elle eklemek zorunda kalırsınız — bu da tekrar, hata riski ve bakım zorluğu demektir.

---

## 2. Basit Bir Dekoratör Örneği: Fonksiyon Süresini Ölçme

En temel decorator yapısını görelim:

```python
import time

def timer(func):
    """Bir fonksiyonun çalışma süresini ölçen decorator."""
    def wrapper(*args, **kwargs):
        baslangic = time.time()
        sonuc = func(*args, **kwargs)
        sure = time.time() - baslangic
        print(f"'{func.__name__}' fonksiyonu {sure:.4f} saniye sürdü.")
        return sonuc
    return wrapper

@timer
def yavas_fonksiyon():
    """Biraz bekleyen örnek bir fonksiyon."""
    time.sleep(1.5)
    return "İşlem tamam!"

print(yavas_fonksiyon())
```

**Çıktı:**
```
'yavas_fonksiyon' fonksiyonu 1.5012 saniye sürdü.
İşlem tamam!
```

### Nasıl Çalışıyor?

```
yavas_fonksiyon = timer(yavas_fonksiyon)
```

`@timer` yazdığımızda Python şunu yapar: `yavas_fonksiyon = timer(yavas_fonksiyon)`. Yani `yavas_fonksiyon` artık orijinal fonksiyon değil, `timer`'ın döndürdüğü `wrapper` fonksiyonudur. `wrapper` içinde önce süre başlatılır, asıl fonksiyon çağrılır, süre hesaplanıp yazdırılır ve sonuç döndürülür.

## 3. `functools.wraps` — Metadata'yı Koruma

Dekoratör kullandığımızda orijinal fonksiyonun `__name__`, `__doc__` gibi bilgileri kaybolur:

```python
print(yavas_fonksiyon.__name__)  # 'wrapper'
print(yavas_fonksiyon.__doc__)   # None
```

Bunu düzeltmek için `functools.wraps` kullanırız:

```python
import time
from functools import wraps

def timer(func):
    @wraps(func)  # Bu satır, wrapper'ın metadata'sını func'tan kopyalar
    def wrapper(*args, **kwargs):
        baslangic = time.time()
        sonuc = func(*args, **kwargs)
        sure = time.time() - baslangic
        print(f"'{func.__name__}' {sure:.4f} saniye sürdü.")
        return sonuc
    return wrapper

@timer
def yavas_fonksiyon():
    """Biraz bekleyen örnek bir fonksiyon."""
    time.sleep(1)

print(yavas_fonksiyon.__name__)  # 'yavas_fonksiyon'
print(yavas_fonksiyon.__doc__)   # 'Biraz bekleyen örnek bir fonksiyon.'
```

**Kural:** Her decorator yazdığınızda `@wraps` kullanın. Bu iyi bir alışkanlıktır ve hata ayıklamayı kolaylaştırır.

## 4. Parametre Alan Dekoratörler

Bazen decorator'ın kendisinin de parametre almasını isteriz. Örneğin, tekrarlama sayısını belirleyebileceğimiz bir `repeat` decorator'ı:

```python
from functools import wraps

def repeat(n):
    """Bir fonksiyonu belirtilen sayıda tekrarlayan decorator."""
    def decorator(func):
        @wraps(func)
        def wrapper(*args, **kwargs):
            for i in range(n):
                sonuc = func(*args, **kwargs)
            return sonuc
        return wrapper
    return decorator

@repeat(n=3)
def merhaba_de():
    print("Merhaba!")

merhaba_de()
```

**Çıktı:**
```
Merhaba!
Merhaba!
Merhaba!
```

Burada iki katmanlı bir yapı var:
1. `repeat(n)` — dıştaki fonksiyon, parametreyi alır
2. `decorator(func)` — asıl decorator
3. `wrapper(*args, **kwargs)` — sarma fonksiyonu

> `@repeat(n=3)` aslında `merhaba_de = repeat(n=3)(merhaba_de)` anlamına gelir.

## 5. Birden Fazla Dekoratör (Stacking)

Bir fonksiyona birden fazla decorator uygulayabiliriz. Sıralama önemlidir — en yakındaki decorator önce uygulanır:

```python
from functools import wraps

def bold(func):
    @wraps(func)
    def wrapper():
        return f"<b>{func()}</b>"
    return wrapper

def italic(func):
    @wraps(func)
    def wrapper():
        return f"<i>{func()}</i>"
    return wrapper

@bold
@italic
def mesaj():
    return "Python Decorator"

print(mesaj())  # <b><i>Python Decorator</i></b>
```

Uygulama sırası: `mesaj = bold(italic(mesaj))`

## 6. Sınıf Tabanlı Dekoratörler

Dekoratörler sadece fonksiyon değil, sınıf kullanılarak da yazılabilir. Bunun için `__call__` metodu kullanılır:

```python
from functools import wraps

class Tekrarla:
    def __init__(self, n):
        self.n = n

    def __call__(self, func):
        @wraps(func)
        def wrapper(*args, **kwargs):
            for _ in range(self.n):
                sonuc = func(*args, **kwargs)
            return sonuc
        return wrapper

@Tekrarla(n=2)
def selam():
    print("Selam!")

selam()
```

Sınıf tabanlı decorator'lar **durum (state) tutmanız** gerektiğinde avantajlıdır — örneğin bir fonksiyonun kaç kez çağrıldığını saymak:

```python
class Sayac:
    def __init__(self, func):
        self.func = func
        self.kac_kere = 0

    def __call__(self, *args, **kwargs):
        self.kac_kere += 1
        print(f"'{self.func.__name__}' {self.kac_kere}. kez çağrıldı.")
        return self.func(*args, **kwargs)

@Sayac
def test():
    print("Test çalıştı.")

test()  # 'test' 1. kez çağrıldı.
test()  # 'test' 2. kez çağrıldı.
```

## 7. Gerçek Hayattan Örnekler

### a) Yetkilendirme (Authorization)

```python
from functools import wraps

def admin_gerekiyor(func):
    @wraps(func)
    def wrapper(kullanici, *args, **kwargs):
        if not kullanici.get("admin"):
            raise PermissionError("Bu işlem için admin yetkisi gerekli!")
        return func(kullanici, *args, **kwargs)
    return wrapper

@admin_gerekiyor
def kullanici_sil(kullanici, kullanici_id):
    print(f"Kullanıcı {kullanici_id} silindi.")

# admin_kullanici = {"isim": "Ali", "admin": True}
# kullanici_sil(admin_kullanici, 42)  # Çalışır

# normal_kullanici = {"isim": "Ayşe", "admin": False}
# kullanici_sil(normal_kullanici, 42)  # PermissionError!
```

### b) Loglama

```python
from functools import wraps
import logging

logging.basicConfig(level=logging.INFO)

def logla(func):
    @wraps(func)
    def wrapper(*args, **kwargs):
        logging.info(f"Çağrılıyor: {func.__name__}(args={args}, kwargs={kwargs})")
        sonuc = func(*args, **kwargs)
        logging.info(f"Döndü: {func.__name__} -> {sonuc}")
        return sonuc
    return wrapper

@logla
def topla(a, b):
    return a + b

print(topla(3, 5))
```

### c) Önbellekleme (Memoization/Caching)

```python
from functools import wraps

def memoize(func):
    cache = {}
    @wraps(func)
    def wrapper(*args):
        if args in cache:
            print(f"Önbellekten alındı: {args}")
            return cache[args]
        sonuc = func(*args)
        cache[args] = sonuc
        print(f"Hesaplandı ve kaydedildi: {args}")
        return sonuc
    return wrapper

@memoize
def fibonacci(n):
    if n < 2:
        return n
    return fibonacci(n - 1) + fibonacci(n - 2)

print(fibonacci(10))  # Özyinelemeli çağrılar da önbellekte!
```

## 8. Dekoratörler İçin Hızlı Şablon

Her decorator yazarken kullanabileceğiniz evrensel şablon:

```python
from functools import wraps

def benim_decoratorum(func):
    @wraps(func)
    def wrapper(*args, **kwargs):
        # Fonksiyon ÇAĞRILMADAN ÖNCE yapılacaklar
        # ...
        sonuc = func(*args, **kwargs)
        # Fonksiyon ÇAĞRILDIKTAN SONRA yapılacaklar
        # ...
        return sonuc
    return wrapper
```

## Özet

| Kavram | Açıklama |
|--------|----------|
| **Decorator** | Başka bir fonksiyonu saran, onun davranışını değiştiren fonksiyon |
| **`@sembol`** | Dekorator uygulama sözdizimi (`@timer` = `func = timer(func)`) |
| **`*args, **kwargs`** | Her türlü argümanı yakalamak için kullanılır |
| **`@wraps`** | Orijinal fonksiyonun metadata'sını (ad, docstring) korur |
| **İç içe decorator** | `@bold @italic` = `bold(italic(func))` şeklinde çalışır |

Dekoratörler Python'ın en zarif özelliklerinden biridir — tekrarı azaltır, kodu daha okunabilir yapar ve **cross-cutting concerns** (loglama, yetkilendirme, caching gibi) sorunlarını temiz bir şekilde çözer.
