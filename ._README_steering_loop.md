# ReYMeN Steering Loop — 5 Katmanlı ReAct Sarmalayıcı

## Ne İşe Yarar

`alt_ajan.py` / `motor.py` / `beyin.py` etrafındaki steering loop'u kurar.
Gözlem → Kanca → Motor → Hafıza → Talimat sırasıyla çalışır.

## 5 Katman

### K1 — hafiza.py (Kalıcı Görev Hafızası)
- Task ID bazlı JSON dosyaları → `.alt_ajan_hafiza/` dizini
- Her adım/sonuç/hata kaydı diske yazılır
- Process crash'e dayanıklı (checkpoint mantığı)
- 72 saat otomatik temizleme

### K2 — Sandbox (Thread Havuzu)
Mevcut: `ThreadPoolExecutor` ile thread izolasyonu
Eksik: Docker/subprocess sandbox. Mimari karar bekliyor.

### K3 — Sistem Talimatı
Mevcut: `sistem_talimati_olustur()` → zaten yeterli

### K4 — kancalar.py (Eylem Öncesi Kural Denetimi)
- Aynı eylem 4 kere art arda → task bloke (döngü koruması)
- Yasaklı araçlar: `ALT_AJAN_GOREVLENDIR` (zincir koruması)
- Task başına 30 eylem limiti
- Hızlı döngü kontrolü (0.5s minimum aralık)
- Derinlik > 3 uyarısı
- `bloke_coz(task_id)` ile manuel kurtarma

### K5 — gozlem.py (LLM Maliyet/Süre Gözlemi)
- Her LLM çağrısının token, süre, maliyeti task_id bazlı
- Provider fiyat tablosu (deepseek, openai, anthropic, groq, lmstudio, ollama)
- `task_ozet()` → tek task raporu
- `genel_ozet()` → tüm sistem maliyet dökümü
- Disk log `.alt_ajan_gozlem/gozlem_log.txt`

## Entegrasyon Noktaları

alt_ajan.py zaten `try/except` ile import ediyor:
```python
from hafiza import alt_ajan_hafiza as _hafiza
from gozlem import gozlemci as _gozlemci
from kancalar import kanca_motoru as _kanca
```

Kanca, `alt_ajan.py`'de `calistir()` içinde `_kanca.denetle()` çağrısı ile çalışır.
Gözlem, `_llm_uret_timeout()` içinde `_gozlemci.kaydet()` çağrısı ile çalışır.
Hafıza, her adım/sonuç/hata sonrası `_hafiza.kaydet()` çağrısı ile çalışır.
